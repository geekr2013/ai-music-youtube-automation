# Free Generation Strategy

## 검토한 Kaggle 예시

### Music Recommendation System using Spotify Dataset

- 목적: 사용자의 취향과 곡 특성을 바탕으로 음악을 추천하는 분석/추천 시스템입니다.
- 한계: 음악 파일을 직접 생성하지 않습니다.
- 반영한 점: 자동 생성 시 장르를 무작위로만 고르지 않고, 10대/20대에게 맞는 장르 후보에 점수를 부여해 선택하도록 반영했습니다.

### Music Generation: LSTM

- 목적: MIDI 또는 음표 시퀀스를 학습해 다음 음표를 예측하는 방식입니다.
- 한계: 기본 LSTM 생성물은 전자음악 완성곡 품질까지 바로 이어지기 어렵고, 학습 데이터의 라이선스 확인이 필요합니다.
- 반영한 점: LSTM 노트북 코드를 직접 복사하지 않고, 반복 패턴/멜로디/섹션 구조 아이디어만 참고했습니다.

## 현재 적용 방식

현재 기본 생성기는 아래 원칙을 따릅니다.

- 유명 아티스트/곡/보컬을 모방하지 않습니다.
- 외부 유료 API를 쓰지 않습니다.
- 저작권 위험이 있는 샘플을 사용하지 않습니다.
- Kaggle 무료 Notebook에서 실행됩니다.
- 생성 후 영상/이미지/음원 임시 파일을 정리합니다.

## 품질 개선 내용

- 장르 선택: hyperpop, electropop, UK garage, drum and bass-lite, dreamy electronic pop 중심
- 음악 구조: 인트로, 빌드업, 후렴, 아웃트로 느낌의 에너지 변화
- 사운드 구성: 킥, 스네어, 하이햇, 베이스, 패드, 아르페지오, 후렴 리드
- 영상 구성: 배경 이미지 8장 자동 생성, 천천히 줌인되는 영상으로 렌더링
- 메타데이터: 제목, 설명, 해시태그 자동 생성
